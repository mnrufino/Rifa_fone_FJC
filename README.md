# Rifa Beneficente - 200 Números

Página web simples para rifa com 200 números.

## Recursos

- Seleção de vários números para a mesma pessoa
- Números de 001 a 200
- Azul para números selecionados
- Vermelho para números vendidos
- Salva os números vendidos no navegador usando `localStorage`
- Envio da participação pelo WhatsApp
- Botão para limpar seleção
- Botão para limpar todos os números vendidos

## Arquivos

- `index.html`
- `style.css`
- `script.js`

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie os arquivos `index.html`, `style.css`, `script.js` e `README.md`.
3. Vá em `Settings`.
4. Entre em `Pages`.
5. Em `Branch`, selecione `main`.
6. Clique em `Save`.

Depois disso, o GitHub irá gerar um link público para sua página.
